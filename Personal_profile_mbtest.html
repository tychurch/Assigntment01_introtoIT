The Myers-Briggs Test

The Myer-Briggs Test is a physcologist personality test designed to identify a personality type.

The results of this test in short mean im a dependable personal with good concentration capable of accomplishing what i set my mind to.

In a team some of these traits can be used to describe the way a person works and how they work, whether it is a mind set where they are heavily focused or if they would rather get the job done as quick as possible.

When taking this into account for forming a team this can be used to organise which personality traits correspond best with others for example when assigning roles some personality traits may fit certain roles better or for some projects if they are all the same personality trait they may work together to be thorough all the way through or to finish quick.
